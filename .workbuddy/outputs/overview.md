## 新页面结构

```
设置 → 隐私设置（一级）
       ├── 学校信息可见性（二级）
       ├── 搜索与推荐（二级）
       ├── 互动权限（二级）→ 仅私信 + 关注 + 评论说明
       └── 更多设置（二级）→ 仅匿名消息开关

设置 → 内容可见性（独立入口）
       ├── 我的帖子（4级单选框）
       ├── 我的评论（4级单选框）
       └── 我的收藏（4级单选框）

发帖页 → 实时权限控制
       ├── 所有人可见 / 仅关注者 / 仅互关者 / 仅自己
       ├── 仅本校可见（switch）
       └── 匿名发布（switch）
```
